** 本文将详细介绍如何在安卓手机上搭建Phira-mp多人联机服务器，让你随时随地和朋友一起玩Phira **

# 软件安装以及准备

我们只需要准备一个软件即可，那就是由hanxinhao000维护的ZeroTermux。

## 下载ZeroTermux

ZeroTermux的下载链接是：

- Github：[https://github.com/hanxinhao000/ZeroTermux/releases](https://github.com/hanxinhao000/ZeroTermux/releases)
- ZeroTermux官网：[https://zerotermux.dev/](https://zerotermux.dev/)

这两个选择其一即可。

## 软件准备

### 保活设置

安装完成后我们给软件后台上锁以及忽略电池省电策略。

![保活设置](/assets/20260123/1.jpg)

### 换源

打开软件同意协议之后按下设备的"音量+"或者"双击屏幕左侧"呼出侧边栏。

依次选择"切换源"，点击"清华源"，点击"确定"。

![换源操作](/assets/20260123/2.jpg)

当出现询问[Y/n]时，直接按下回车即可。

![确认换源](/assets/20260123/3.jpg)

### 下载MT管理器

MT管理器下载地址是：

- 官网：[https://mt2.cn/download/](https://mt2.cn/download/)

### 配置MT管理器

我们打开MT管理器，依次点击：

左上角三个横杠 → 侧边栏右上角三个点 → 点击"添加本地存储" → 点击左上角三个横杠 → 选择"ZeroTermux" → 点击下方"适用此文件夹" → 点击"确定"

![MT管理器配置](/assets/20260123/8.jpg)

# 准备ChmlFrp内网穿透

ChmlFrp的官网地址为：[https://panel.chmlfrp.net/](https://panel.chmlfrp.net/)

打开后我们注册一个账号并且登陆，根据提示进行实名认证。实名完成后我们就可以来开通服务了。

### 进入隧道管理

点击左上角三条横杠选择"隧道管理"。

![隧道管理](/assets/20260123/10.jpg)

### 添加隧道

我们点击"添加隧道"，选择一个国内线路如"湖北十堰-2"，选择后点击"继续"。

![添加隧道](/assets/20260123/11.jpg)

### 填写配置

填写一个备注，名字随意也可以默认。内网端口填写"12346"。

![填写配置](/assets/20260123/12.jpg)

### 获取配置代码

点击刚创建的隧道右下角扳手，点击"获取配置代码"。

![获取配置代码](/assets/20260123/13.jpg)

点击"Frpc.ini配置"，再点击左下方"复制配置"，把复制下来的东西粘贴到其他地方保存起来。

![复制配置](/assets/20260123/14.jpg)

### 复制服务器地址

在隧道列表中点击"连接地址:xxxx"即可复制服务器的地址。

![复制服务器地址](/assets/20260123/15.jpg)

# 安装Phira-mp以及安装ChmlFrp

温馨提示：如果下载失败请尝试切换网络环境

### 安装Phira-mp

在ZeroTermux中输入以下命令回车：

```bash
curl -LO https://riluo-ya.github.io/blog/assets/20260123/install.sh && bash install.sh
```

当出现"Phira多人联机一键安装脚本"的提示就说明成功了。

![安装脚本](/assets/20260123/4.jpg)

接下来输入"1"回车。如果当出现询问[Y/n]时，直接按下回车即可。

当提示"安装全部完成！"时就代表Phira-mp已经安装完成。

![安装完成](/assets/20260123/5.jpg)

### 安装ChmlFrp内网穿透

等待Phira-mp安装完成后再次输入命令回车：

```bash
curl -LO https://riluo-ya.github.io/blog/assets/20260123/install.sh && bash install.sh
```

在接下来的界面选择"2"回车。

![选择安装项](/assets/20260123/4.jpg)

当出现"提示：建议用文本编辑器打开frpc.ini进行配置"时我们打开MT管理器（注意不要关闭ZeroTermux）。

![配置提示](/assets/20260123/6.jpg)

在MT管理器点击左上角三个横杠，选择刚刚添加的"Termux Home"，进入"ChmlFrp-0.51.2_251023_linux_arm64"文件夹，点击"frpc.ini"文件。

![打开配置文件](/assets/20260123/9.jpg)

把里面的内容删掉换成我们之前复制的配置代码，保存退出即可。

我们回到ZeroTermux按下回车，当出现"[成功] 内网穿透工具安装完成！ (≧∇≦)ﾉ"的提示即为安装完成。

![内网穿透安装完成](/assets/20260123/7.jpg)

# 启动服务器

我们打开ZeroTermux会出现"Phira 多人联机启动器"的提示。

![启动器界面](/assets/20260123/16.jpg)

我们输入"1"回车启动Phira-mp。当出现"Local Address: [::]:12346"的字样就代表启动成功。

![服务器启动成功](/assets/20260123/17.jpg)

接下来我们按下设备的"音量-"按键打开侧边栏，点击"切换会话"再点击"新会话"。

![新建会话](/assets/20260123/18.jpg)

我们再按下"音量+"回到新开的会话页面，再选择"2"回车，出现"映射启动成功, 感谢您使用ChmlFrp!"即为启动成功。

![内网穿透启动成功](/assets/20260123/19.jpg)

# 最后

最后！我们只需要回到游戏里把之前复制的服务器链接粘贴上就可以玩了！

![游戏内连接](/assets/20260123/20.jpg)

每次要联机的时候只需要重复"四.启动服务器"的步骤即可！
